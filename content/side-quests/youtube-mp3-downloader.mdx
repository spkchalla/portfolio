# YouTube MP3 Downloader

A simple, fast, and high-quality YouTube to MP3 downloader.

## Features

- **High Quality**: Downloads audio in the best possible bitrate.
- **Fast Conversion**: Uses `yt-dlp` for efficient extraction and `ffmpeg` for conversion.
- **Clean UI**: A minimalist React-based frontend for a seamless experience.
- **Local First**: Designed to run easily on your own machine.

Check out the [GitHub repository](https://github.com/spkchalla/youtube-mp3-downloader) for installation instructions and source code.
